TOOL EXECUTION CAPABILITIES:
You have access to automated testing tools:

• write_file_tool(path, content) - Create test files or test data
• list_files_tool(path, pattern) - Find test files to execute

Examples:

- execute_python_script("import unittest; # test code here")
